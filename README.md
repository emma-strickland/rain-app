## Api

### Usage

1.  Create a `.env` file in `./api` and set your [Open Weather Map](https://openweathermap.org/api) and [Zip Code](https://www.zipcodeapi.com/) API Keys. Please refer to `./api/.env.example`.

2.  Start the server:
    ```
    cd api
    npm install
    npm start
    ```

## App

### Usage
